概要

Sparkplugでは、JavaScriptを使ってコンポーネントにリッチで動きのあるインタラクションを追加しています。
利用にあたっては、あらかじめ「sparkplug.js」をはじめとするJSファイル一式を読み込んでおく必要があります。
すべてのページテンプレートは、はじめからこれらのJSファイル一式が読み込まれた状態になっています。

sparkplug-*.*.*/
  └── js/
      |── bootstrap.js
      |── jquery-*.*.*.js
      |── sparkplug.js
      |── ...

data属性から設定

.sp-datagrid が付与されている要素に data-sp-toggle="sp-datagrid" と指定します。

data-sp-toggle-{オプション名}={value} と指定することで、オプション値を設定することも可能です。

ただし、動的にデータグリッドの行を追加した場合、data属性からの設定では正常に動作しませんので、 その場合はJavaScriptから設定を行うようにしてください。

JavaScriptから設定

spDatagrid メソッドを使用します。オプションはメソッドのパラメータにオブジェクトリテラルで指定します。

また、パラメータに直接 false を渡すことで、機能を無効にすることも可能です。

ツールチップ

data-toggle="tooltip" を付与するとツールチップが自動で有効になります。 追加でJavaScriptのコードを実行する必要はありません。ツールチップはデフォルトで、マウスオーバーしてから0.5秒（500ms）後に表示、マウスアウトの0.1秒（100ms）後に非表示となります。

Polyfill

SparkplugではHTML5やCSS3の機能を使用していますが、これらに未対応のブラウザ(主に旧バージョンのIE)でも最新の対応ブラウザ同様に表示／動作する仕組み(Polyfill）を提供しています。

autofocus属性の指定を有効にする（IE8-9向け）

HTML5のautofocus属性に対応していないIE8-9でも、autofocusを指定した要素がページロード時に自動的にフォーカスされるようになります。
対象の要素にautofocus属性を付与するだけで、特に設定は必要ありません。

placeholder属性の指定を有効にする（IE8-9向け）

HTML5のplaceholder属性に対応していないIE8-9でも、テキストボックス未入力時のみ表示されるヒントテキストが有効になります。
対象の要素にplaceholder属性を付与するだけで、特に設定は必要ありません。
