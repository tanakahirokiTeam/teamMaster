CSSクラス名とBootstrapの関係について

ページテンプレートやコンポーネントのHTMLで使われているCSSクラス名のうち、.sp- で始まるクラスはすべてSparkplugオリジナル（独自）のスタイルです。

一方、.sp- が付いていないクラスはすべて Bootstrap のスタイルですので、クラスの追加／削除によってスタイルを微調整する場合は、Bootstrapのドキュメント（Bootstrap CSS, Bootstrap Components）もあわせて確認してください。

ソート可能な列の表現

データグリッドにソート（並べ替え）機能を持たせる場合、対象列のth要素に識別のための専用class .sp-col-sortable を追加します。th内には、列のソート順を示すアイコンつきのリンクを記述してください。
現在ソート対象となっている列には、th要素に強調表示用class .sp-col-sorted を追加します。
