以下のSparkplug利用規約（以下「本規約」という。）をご確認ください。
本規約に同意できない場合、本ソフトウェアを使用することはできません。本ソフトウェアを使用した場合、本規約に同意したものとみなされます。

Sparkplug 利用規約

第1条（定義）

  1. 「本ソフトウェア」とは、株式会社電通国際情報サービス（以下「当社」という。）を開発元とするCSSフレームワーク「Sparkplug」であり、機械により読み取りうる形で提供されるコンピュータ・プログラム（以下「本プログラム」という。）及びこれを使用するために当社が提供するユーザ・マニュアル等の資料（以下「関連資料」という。）を意味します。
  2. 「使用」とは、本プログラムの全部または一部を機械に読みこませること、転送することまたは実行すること、実行結果を出力すること、および関連資料を利用すること、ならびに本プログラムを改変し、改変した本プログラムについて前記の行為を行うことを意味します。

第2条（本ソフトウェアの使用）

  1. 当社は、ユーザ（ユーザがその所属する法人等の団体の業務のために本ソフトウェアを使用する場合には、当該法人等の団体を意味し、以下「ユーザ」という。）が本規約を遵守することを条件として、ユーザに対し、本ソフトウェアを提供し、第7条に定める有効期間中、本ソフトウェアを使用するための非独占的かつ譲渡不能の使用権を無償にて設定します。
  2. ユーザは、当社より受託した業務（以下「目的業務」という。）を遂行する目的にのみ本ソフトウェアを使用することができるものとします。
  3. ユーザは、本ソフトウェアの全部または一部を逆コンパイル、逆アセンブル、その他リバースエンジニアリング等することはできません。また、有償・無償を問わず、本ソフトウェアの全部または一部を第三者に譲渡、再使用許諾または貸与することはできません。
  4. ユーザは、第7条に定める有効期間終了後速やかに本ソフトウェアを破棄または当社に返却し、あわせてユーザの機械上に複製した本ソフトウェアを消磁するものとします。

第3条（守秘義務）

  1. ユーザは、本ソフトウェア並びにそれに利用されている方法及び概念（以下「秘密情報」という。）について善良なる管理者の注意をもって管理するものとし、当社の書面による明示の承認を得ずに、第三者に開示・漏洩してはなりません。以下の情報は、秘密情報とは見做されません；(i) 本規約の違反によらず、一般に入手可能な情報、(ii)いかなる守秘義務にも違反せずに第三者から正当に入手した情報、(iii) ユーザが秘密情報とは無関係に独自に開発した情報、または(iv) 開示時点でユーザが既に知っていた情報。
  2. 本条の規定は、第7条に定める有効期間終了後も有効に存続します。

第4条（保証及び責任）

当社は、本ソフトウェアを現状有姿にてユーザに提供し使用許諾するものであり、本ソフトウェアに瑕疵が存しないこと、本ソフトウェアが第三者の権利を侵害していないこと、本ソフトウェアの機能がユーザの要求を満たすことを含め、明示的であると黙示的であるとを問わず一切保証しません。また、当社は、ユーザによる本ソフトウェアの使用または使用不能から生じる如何なる損害についても責任を負わないものとします。

第5条（知的財産権）

  1. ユーザは、本ソフトウェアが当社の財産であり、かつその一切の知的財産権は当社に帰属していることを了解します。
  2. ユーザは、本ソフトウェアに含まれる当社の著作権表示を変更し、除去しもしくは削除してはなりません。

第6条（OSSの使用条件）

本ソフトウェアと共にユーザに提供されるTwitter Bootstrap等のオープンソースソフトウェア（以下「OSS」という。）については、本規約の定めは適用されず、OSSの提供元が定める使用条件が適用されるものとします。

第7条（有効期間）

  1. 本規約に基づく本ソフトウェアの使用許諾の有効期間は、ユーザが本ソフトウェアの使用を開始した時点から、目的業務が終了するまでとします。
  2. ユーザが本規約のいずれかの条項に違反した場合、前項の定めに拘らず、本規約に基づく本ソフトウェアの使用許諾は直ちに終了するものとします。
  3. 第1項の定めに拘らず、当社はユーザに通知することにより、いつでも本規約に基づく本ソフトウェアの使用許諾を終了することができるものとします。

第8条（一般条項）

  1. 本規約に関して疑義が生じた場合、ユーザ及び当社は信義誠実の原則に従い協議のうえ円満に解決を図るものとします。
  2. 本規約に関し、訴訟の必要が生じた場合には、東京地方裁判所を第一審の専属的合意管轄裁判所とします。
